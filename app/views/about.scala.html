About Shrty

Shrty is an open source URL shortening service that is meant to be locally hosted. Configured in the proper way, it could be scaled to serve high volumes of traffic.

License

Shrty is released under the Apache

Underlying Tech

It is built on the Play Framework and uses a PostgreSQL back-end. The source code is available via GitHub.
